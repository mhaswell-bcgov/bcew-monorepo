# WordPress Monorepo Documentation

This site is the central docs portal for all plugins, themes, and shared monorepo workflows.

## How docs are organized

- Monorepo-level documentation lives in this root `docs/` folder.
- Plugin and theme docs stay close to their code in `plugins/<slug>/docs/` and `themes/<slug>/docs/`.
- A sync script copies package docs into this site before running VitePress.
