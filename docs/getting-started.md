# Getting Started

Run all commands on this page from the monorepo root:
`/Users/aspiteri/Development/wordpress-monorepo`

## Prerequisites

- Node version that matches the repository `engines.node`
- `pnpm` installed globally

## Install

```bash
# from monorepo root
pnpm install
```

## Run docs locally

```bash
# from monorepo root
pnpm docs:dev
```

## Build docs

```bash
# from monorepo root
pnpm docs:build
```

## Preview built docs

```bash
# from monorepo root
pnpm docs:preview
```
