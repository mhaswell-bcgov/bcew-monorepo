Media & Text Layout ​

Presents media and text side-by-side with configurable image placement.

Notes ​

  • Block name: bcew-blocks/media-text-layout
  • Category: media
  • Intended use: paired media and narrative content sections

Attributes ​

  • imageId (number): selected media library image ID
  • imagePosition (string, default left): media placement relative to text

Authoring Guidance ​

  • Keep text concise when media is the primary focus.
  • Use image placement to support reading flow and hierarchy.
